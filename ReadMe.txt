#Teste Desenvolvedor

##Projeto de cadastro de vendas com cashback

===========================

* Requisitos

Visual Studio

SQL Server

Postman ou Swagger



* Instalação

Extrair arquivos em um mesmo diretório

Restaurar Banco de dados a partir do arquivo Dados.bat

Abrir o projeto TesteDesenvolvedor.sln

Alterar o Server em appsettings.json para a instancia do servidor local de SQL Server



* Como usar

Compilar usando Swagger ou Postman para testes

Para Buscar Cervejas, informar a página e a quantidade de itens a serem mostrados por página, com
limite de itens entre 1 e 100

Para Buscar Cervejas por Id, informar Id

Para Buscar Vendas, informar a página e a quantidade de itens a serem mostrados por página, com
limite de itens entre 1 e 100, se desejar alterar os filtros de busca por data mínima e máxima

Para Buscar Vendas por Id, informar Id

Para Inserir Vendas, informar(alterando apenas) os campos quantidadeItem(quantidade de cervejas) 
e cervejasId(cerveja)
